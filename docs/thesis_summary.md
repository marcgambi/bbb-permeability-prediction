# Thesis Summary

## Title

Prediction of Blood-Brain Barrier Permeability Using Machine Learning and Graph Neural Networks

## Objective

The aim of this project is to predict the logarithmic blood-brain barrier partition coefficient (logBB) from molecular structures.

## Implemented Models

- Extra Trees Regressor
- Sequential Feature Selection
- SchNet
- Multi-seed SchNet Ensemble
- Hybrid SchNet + Extra Trees

## Technologies

- Python
- PyTorch
- PyTorch Geometric
- RDKit
- Scikit-learn

## Evaluation Metrics

- R²
- RMSE
- MAE
- MSE
